# Shannon-Fano-Algorithm
  
Description  
This script realize of the Shannon-Fano data compression algorithm  
  
Requirements  
Python 3.8  
  
Launch:  
shannon-fano.py  
Example: ./shannon-fano.py [code/decode] [INPUT_FILENAME] [OUTPUT_FILENAME]  

Text file format example:  
[tbs]  
table  
[tbe]  
[cts]  
bin code  
[cte]
